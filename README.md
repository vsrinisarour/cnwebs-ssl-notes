# cnwebs-ssl-notes

本仓库用于归档和发布多个独立的 HTML 页面。页面内容主要涉及与网络连接安全相关的观察与记录，不针对任何特定域名或具体网站。

## 项目简介

该仓库旨在以静态页面的形式，整理和展示一系列关于 SSL/TLS 证书、HTTPS 部署以及相关网络连接状态的笔记与参考信息。所有页面均为独立 HTML 文件，可直接在浏览器中打开查看，无需额外依赖或后端服务。

## 目录说明

```
cnwebs-ssl-notes/
├── README.md            # 本文件
├── pages/               # 存放所有 HTML 页面
│   ├── index.html       # 页面索引（如有）
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
└── assets/              # 静态资源（如有）
    ├── css/
    └── images/
```

- `pages/`：主要页面目录，所有发布的 HTML 文件均置于此处。
- `assets/`：页面可能引用的样式、图片等静态资源。

## 页面归档说明

- 每个 HTML 页面均为独立、自包含的文档，内容围绕 SSL/TLS 相关主题展开。
- 归档内容仅为个人学习、记录或技术参考用途，不保证时效性或完整性。
- 页面可能包含示例性描述、代码片段或配置参考，但均不指向任何实际运行中的服务或网站。
